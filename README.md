# hangmanCLI

A command line version of hangman using node.js

This project was an exercise in using object constructors, using export/requires, and game logic. 

## Download Locally

Clone the repo  

```
git clone https://github.com/zaynaib/hangmanCLI.git
```

Terminal commands 

```
npm install

node cli.js

```
